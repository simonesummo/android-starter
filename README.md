# Android Starter

An evolving Android starter template for building modern applications with Kotlin, Jetpack Compose, and the latest Android development tools.

## Goal

This project serves as my personal foundation for new Android applications.

Rather than being a one-time template, it is a living project where I experiment with modern Android development practices, evaluate new libraries, and refine architectural decisions over time.

### The main goals are to:

* Keep up with the Android ecosystem
* Explore new technologies and best practices
* Build a reusable foundation for future projects
* Improve development workflow and tooling
* Document architectural decisions and experiments

### Planned Features

* Kotlin & Jetpack Compose
* Material 3
* Modular architecture
* Dependency Injection
* Navigation
* Networking
* Local persistence
* Unit & UI testing
* GitHub Actions
* Static analysis
* Dependency management
* Release automation

## Project Status

🚧 Work in progress.

This repository evolves continuously as I learn, experiment, and adopt new Android technologies.
